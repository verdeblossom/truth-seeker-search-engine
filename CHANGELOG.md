# Changelog

All notable changes to the Truth-Seeking Search Engine will be documented in this file.

## [v1.1.0] - 2026-04-04
### Changed
- Updated truth-seeking heuristics to prioritize Grokipedia.com (boosted to 20) as a truth-seeking encyclopedia
- Lowered Wikipedia.org weighting (now 8) to reflect preference for more neutral, objective sources
- Applied the same change to both console (`truth-search.py`) and GUI (`truth-search-gui.py`) versions

### Philosophy
The ranking now better aligns with the project's supreme directive: grounding results in objective reality over crowdsourced consensus.

## [v1.0.0] - 2026-04-04
### Added
- Graphical User Interface (GUI) using Streamlit with clickable result links
- Browser-based interface for easier daily research use
- Clean, professional README.md with clear console + GUI instructions

### Changed
- Updated installation and usage sections
- Added requirements.txt for one-command installation

## [v0.3.0] - 2026-04-04
### Added
- v3 truth-seeking heuristics with improved primary-source detection
- Support for the new `ddgs` package
